Modal builder Create modals without any static HTML placeholders

About

This plugin allows you to create modals dynamically, without cluttering your source with static HTML for modals. Everything is created in JavaScript and fully customizable.

Download file

Basic example

Using bootstrap-modal-builder.js

Code for the example above:

$.modal({
  title: "Example title",
  content: "Here's some important message you must read.",
  buttons: [
    {
      text: "Roger that",
      close: true
    }
  ]
}).modal('show');
Note: $.modal() returns the actual HTML node for created modal so you can chain it with Bootstrap's .modal() method to manipulate it.

Options

Name type default description
title string "" Title of the modal in .modal-header.
content string "" Content of the modal, can be a HTML string.
header boolean true Display the .modal-header. If hidden, title and .close button are invisible
footer boolean true Display the .modal-footer. Please note that when false buttons will also not appear.
dispose boolean true Remove the actual HTML element with modal on close. Set to false if you want to reuse the same modal multiple times.
buttons array [] List of buttons to create in the modal. See the table below for button specific options.

Options for buttons

These options apply to objects in buttons array.

Name type default description
text string "" Text visible on the button.
href string "#" Link used in the href attribute.
close boolean false When true, pressing the button will also close the modal. Behind the scenes it adds data-dismiss="modal".
classes string "" List of classes applied to the button HTML tag. For example, if you want a button to be primary and large, set it to: btn-primary btn-large. btn class is applied by default so you don't have to add it here.
html string "" Custom HTML for the button. Use it if your button tag needs more tweaking. Remember that the text, href and classes options take precedence and can still modify the custom HTML. For example additional classes will be appended to the custom element.
click function undefined Callback function fired when a button is clicked.

Full example

Method with all options possible:

$.modal({
  title: "Full example title",
  content: "<p>Rich content with some <strong>HTML</strong> tags <em>here</em> and <u>there</u></p>",
  header: true,
  footer: true,
  dispose: true,
  buttons: [
    {
      text: "Yup, I got it",
      href: "#",
      classes: "btn-primary",
      click: function(e) {
        alert("Seriously, I got it!");
      }
    },
    {
      text: "Get me out of here",
      href: "#",
      close: true
    },
    {
      html: "<a href='#' data-dismiss='modal' class='btn btn-danger'>Danger, custom button",
      text: "Achtung!",
      classes: "btn-large",
      click: function() {
        alert("Custom  button");
      }
    }
  ]
}).modal('show');
